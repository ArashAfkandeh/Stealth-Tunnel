# 🚀 GhostRPC

این برنامه یک تونل ارتباطی فوق‌پیشرفته، بسیار سریع با واسطه‌ Cloudflare بر بستر HTTP/2 (همراه با فرمت‌دهی gRPC) می‌باشد که برای عبور از سخت‌گیرانه‌ترین سیستم‌های فیلترینگ طراحی شده است.

---

## ✨ قابلیت‌های کلیدی و منحصربه‌فرد

### ۱. معماری انتقال سریع و پایدار
ترافیک این تونل از نگاه سیستم‌های مانیتورینگ کاملاً شبیه به وبگردی عادی یا یک درخواست استاندارد gRPC از Cloudflare است. هیچ نوع رمزنگاری اضافه و تکراری (Double Encryption) وجود ندارد تا پرفورمنس کاهش نیابد.

### ۲. پشتیبانی بومی از TCP و UDP
تانل GhostRPC علاوه بر ترافیک TCP، از استریم‌های UDP خام (مثل تماس‌های صوتی و بازی‌ها) بر بستر پروتکل پایدار HTTP/2 به صورت بومی و کاملاً همزمان پشتیبانی می‌کند.

### ۳. دفاع هوشمند در برابر فیلترینگ (Active Probing Defense)
سرور دارای قابلیت استتار (Camouflage) است. اگر ربات‌های فیلترینگ پورت ۴۴۳ سرور شما را پویش کنند یا مسیرهایی غیر از مسیر مخفی تونل را درخواست دهند، برنامه می‌تواند به جای خطای مشکوک، محتوای یک سایت مجاز (Proxy)، یک فایل HTML محلی، یا هر صفحه‌ی ساختگی دیگری را به آن‌ها نمایش دهد.

### ۴. دریافت و تمدید گواهینامه SSL کاملاً خودکار (Zero-Config ACME)
بدون نیاز به نصب ابزارهای اضافی! فقط کافیست توکن Cloudflare خود را در کانفیگ سرور قرار دهید. سیستم به طور خودکار چالش DNS-01 را حل کرده، گواهینامه Wildcard یا دامنه‌ی مستقیم را دریافت می‌کند و پیش از انقضا به صورت خودکار آن را تمدید می‌نماید. تمام گواهی‌ها در پوشه‌ی `certs` کنار خود برنامه ذخیره می‌شوند.

### ۵. مسیریاب هوشمند SNI Multiplexer
شما می‌توانید سرور GhostRPC را به عنوان تنها شنونده روی پورت `443` سرور قرار دهید. برنامه با خواندن پکت TLS ClientHello، نام دامنه (SNI) را استخراج می‌کند:
- اگر ترافیک مربوط به سرویس دیگری باشد، اتصال **بدون بازگشایی رمزنگاری** مستقیماً به آن سرویس پاس داده می‌شود.
- اگر ترافیک مربوط به خودِ تونل باشد، برنامه خودش رمزنگاری را باز کرده و آن را وارد تونل می‌کند.

### ۶. بارگذاری مجدد بدون قطعی مطلق (Zero-Downtime Hot-Reload)
تنظیمات تغییر کرد؟ کافیست فایل کانفیگ را ذخیره کنید. برنامه به صورت خودکار تغییرات را می‌فهمد و ترافیک جدید را روی کانفیگ جدید اعمال می‌کند در حالی که کانکشن‌های قدیمی تا لحظه‌ی آخر باز می‌مانند (بدون حتی یک میلی‌ثانیه قطعی).

- لینوکس

## ⚙️ نمونه تنظیمات (Configuration)

### کانفیگ سرور (`server.toml`)
```toml
[server]

# آدرس و پورتی که سرور روی آن گوش می‌دهد
bind_addr = "0.0.0.0:443"

# رمز عبور مشترک بین سرور و کلاینت برای احراز هویت
secret = "my_super_secret_password"

# مسیر مخفی برای درخواست‌های تونل (باید با کلاینت یکی باشد)
hidden_path = "/api/v4/telemetry"

# مسیر فایل‌های گواهینامه TLS (برای ارتباط امن H2/HTTPS)
tls_cert = "fullchain.pem"
tls_key = "privkey.pem"

# هدف استتار (Active Probing Defense) - می‌تواند سایت یا مسیر فایل محلی باشد
camouflage_target = "https://www.ubuntu.com"

# تنظیمات دریافت خودکار گواهینامه Let's Encrypt (DNS-01 Challenge)
cloudflare_api_token = "YOUR_CLOUDFLARE_API_TOKEN"

acme_domains = [
    "domain.com",
    "*.domain.com"
]

# (اختیاری) سطح لاگ‌گیری سیستم: "trace", "debug", "info", "warn", "error"
log_level = "info"

# تنظیمات مسیریابی فوق‌سریع ترافیک بر اساس دامنه (SNI Routing)
# در این بخش می‌توانید تعیین کنید که اگر کلاینت با یکی از دامنه‌های زیر به سرور متصل شد،
# ترافیک آن بدون بازگشایی رمزنگاری (Zero-Copy) مستقیماً به آی‌پی و پورت مشخص شده (مثلاً سرور Xray) هدایت شود.
# فرمت: "دامنه:پورت_ورودی=آی‌پی_مقصد:پورت_مقصد"
port_mappings = [
    "fi.SNI.COM:443=127.0.0.2:443",
    "fr.SNI.COM:443=127.0.0.2:443",
    "tr.SNI.COM:443=127.0.0.2:443",
    "de.SNI.COM:443=127.0.0.2:443",
    "nl.SNI.COM:443=127.0.0.2:443",
    "uk.SNI.COM:443=127.0.0.2:443",
]
```

### کانفیگ کلاینت (`client.toml`)
```toml
[client]

# مسیر مخفی تونل (باید دقیقا مشابه سرور باشد)
hidden_path = "/api/v4/telemetry"

# رمز عبور تونل (باید دقیقا مشابه سرور باشد)
secret = "my_super_secret_password"

# (اختیاری) تعداد کانکشن‌های باز در هر نود برای اتصال سریع‌تر
pool_size_per_node = 16

# پشتیبانی بومی از UDP: اگر yes باشد، برای تمام پورت‌های مپ شده، سوکت UDP هم باز می‌شود
accept_udp = "yes"

# لیست IPهای تمیز کلودفلر (ارتباطات به صورت مسابقه‌ای و هوشمند با سریع‌ترین IP برقرار می‌شود)
clean_ip = [
    "104.21.32.1",
    "172.67.19.1"
]

# (اختیاری) سطح لاگ‌گیری سیستم: "trace", "debug", "info", "warn", "error"
log_level = "info"

# سرورهای لوکیشن NL
[[client.remotes]]
location = "NL"
sni = [
    "sni1.example.com:443",
    "sni2.example.com:443"
]
port_mappings = [
    "sni.nl.example.com:443=127.0.0.1:443",
    "1373=1373"
]

# سرورهای لوکیشن DE
[[client.remotes]]
location = "DE"
sni = [
    "sni3.example.com:443"
]
port_mappings = [
    "sni.de.example.com:443=127.0.0.1:443"
]
```

## 🚀 اجرا
برای نصب به عنوان سرویس دائمی می‌توانید از فایل `GhostRPC.service` استفاده کنید.

### دستورات سریع:
```bash
# سمت سرور
./GhostRPC server.toml

# سمت کلاینت
./GhostRPC client.toml
```
